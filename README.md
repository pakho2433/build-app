# 我的私人記錄

一個只供自己日常使用的本機優先網頁 App。

## 私隱設計

- 個人記錄只儲存在目前裝置的瀏覽器 IndexedDB。
- 記錄以使用者密碼透過 PBKDF2 及 AES-GCM 加密。
- 不使用 Firebase、Supabase、分析服務或任何後端。
- 不會把記錄寫入 GitHub。
- 公開 GitHub 儲存庫只會公開程式碼，不會公開使用者輸入的內容。
- 備份檔仍是加密資料，還原時需要原本密碼。

## 功能

- 密碼建立及解鎖
- 新增、編輯、刪除及搜尋日常記錄
- 本機自動加密儲存
- 匯出及匯入加密 JSON 備份
- 清除所有本機資料
- PWA 主畫面安裝及基本離線使用

## 重要事項

密碼不會儲存，忘記後無法解密資料。刪除 Safari 網站資料、重設瀏覽器或更換手機前，請先匯出備份。
